# Project-P
Egyetemi feladatom volt a 2.féléves Web-Fejlesztés kurzuson.
Feladatom célja egy Pinteresthez hasonló weboldalt készíteni. Ami kezeli a felhasználókat, nem adatbázisban hanem egy txt fájlba.
Éles használatra is sor került, végül barátaim saját szorakoztatásának áldozata lett. Figyelem a weboldalon NSFW tartalom található!
